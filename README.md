# Learning Vue.js

This workspace contains introductory episodes for learning Vue.js, each as a standalone HTML file:

- **Episode01.html**: Basic Vue.js setup and first steps
- **Episode02.html**: Intermediate concepts and examples
- **Episode03.html**: Advanced Vue.js features and usage

## How to Use

1. Open any episode HTML file in your browser to view the content and interact with Vue.js examples.
2. Each file is self-contained and demonstrates different aspects of Vue.js.

## Requirements

- Modern web browser (Chrome, Firefox, Edge, etc.)
- No additional setup required

## About

This project is for learning and experimenting with Vue.js in a simple, file-based format. Each episode builds on the previous one, but can be used independently.
